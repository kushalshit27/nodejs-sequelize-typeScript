## Setup

setup `.env` to run local:

```
NODE_ENV=development

MYSQL_DATABASE=database_name
MYSQL_USER=your_user
MYSQL_PASSWORD=user_password
MYSQL_ROOT_PASSWORD=root_password
MYSQL_HOST=127.0.0.1
```
